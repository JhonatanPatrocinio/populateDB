# populateDB
Script para Popular DB com dados Fakes

Dependencias para funcionar

Instalar, Criar e Usar VirtualEnviroment
 
 apt-get install python3-venv
 
 python3 -m venv venv
 
 source venv/bin/activate
 
 pip install Faker
